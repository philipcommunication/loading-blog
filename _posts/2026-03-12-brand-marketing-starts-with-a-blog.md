# 브랜드마케팅의 시작 블로그 # 
반갑습니다.
오랜만에 포스팅을 작성하게된 로딩입니다.

 근래에 **바이브 코딩**이 유행하면서 예전에 디자인 역량과 개발 역량의 부족으로 인해 언젠가는 꼭 하겠다 마음먹었던 블로그 리뉴얼을 대대적으로 진행했습니다.

 # 개편이유? # 

 스터디를 하면서 만든 블로그를 개인 브랜딩과 마케팅의 도구로 사용하고싶었는데, 지킬기반이라 아무것도 모르고 어어어.. 하면서 만든 블로그를 최대한 살려서 제 입맛대로 만들어보고 싶었습니다. 
사실 도메인도 너무 맘에 들기도 하고요:) 

# 디자인의 영감? # 

 원했던 디자인은 예전 어릴 때 꼭 갖고 싶었던 미니게임 보이의 디자인을 넣고, 포스팅의 모양을 게임 팩의 디자인으로 그리고 홈페이지 곳곳에 이스터에그를 심어두고 싶었습니다. 사이트 곳곳에 이스터에그를 심어뒀고 추가될 예정입니다.

# 블로그의 활용 방법 # 

근 몇 년 동안 여러 수많은 업체를 컨설팅하고 소위 말하는 돈 안드는 마케팅을 진행하면서 빠짐없이 제안하고 강조 했던게 **블로그마케팅**이였는데, 저 자신을 돌아보게 됐습니다. 그래서 블로그를 통해 개인의 브랜딩 나아가서는 제 비즈니스의 구심점으로 만들어 볼까 합니다.
